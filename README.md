##1.Number Guessing Game 
--Description:
The Number Guessing Game is a simple console-based Java application where the player tries to guess a randomly generated number within a limited number of attempts. The game is interactive and consists of multiple rounds, allowing users to improve their guessing skills while tracking their performance through scores and wins.
The system generates a random number between 1 and 100 at the start of each round. The player is given 5 attempts to guess the correct number. After each guess, the program provides hints indicating whether the guessed number is too high or too low. If the player guesses correctly, they earn points based on the number of attempts remaining.
The game continues for multiple rounds until the player chooses to stop. At the end, the application displays the total number of rounds played, rounds won, and the final score.
--Features:
Random number generation between 1 and 100
Limited attempts (5 per round)
Hint system (Too High / Too Low)
Score calculation based on remaining attempts
Multi-round gameplay
Input validation for correct user entries
Performance tracking (total score and rounds won)
--Technologies Used:
Java (Core Java)
Scanner class for user input
Random class for number generation

##2.Student Marks and Grade Calculator 
--Description:
The Student Marks and Grade Calculator is a simple Java-based console application designed to calculate a student’s total marks, average score, and corresponding grade based on marks entered for three subjects. This project demonstrates basic programming concepts such as user input handling, arithmetic operations, and conditional statements.
The user is prompted to enter the student’s name and marks for three subjects. The program then calculates the total marks (out of 300) and the average percentage. Based on the average marks, a grade is assigned using a predefined grading system ranging from A to F.
The application provides a quick and efficient way to evaluate student performance and display results in a readable format.
--Features:
Accepts student name and marks for three subjects
Calculates total marks and average percentage
Assigns grade based on average score
Simple and user-friendly console interaction
Displays formatted result output
--Technologies Used:
Java (Core Java)
Scanner class for user input

##3.ATM Interface System (Java Console Application)
--Description:
The ATM Interface System is a console-based Java application that simulates the basic functionalities of an Automated Teller Machine (ATM). This project allows users to perform essential banking operations such as checking account balance, depositing money, and withdrawing funds in a secure and interactive way.
The system is built using object-oriented programming concepts, where the BankAccount class manages account details and transactions, and the ATM class provides a user interface to interact with the account. The program ensures proper validation of user inputs and handles cases like insufficient balance and invalid transactions.
This application provides a simple yet effective demonstration of how ATM systems work in real life.
--Features:
Check account balance
Deposit money into the account
Withdraw money with balance validation
Menu-driven user interface
Input validation for transactions
Continuous operation until user exits
--Technologies Used:
Java (Core Java)
Object-Oriented Programming (OOP) concepts
Scanner class for user input

##4.currency_converter
--Description
The Currency Converter is a simple Java-based console application that allows users to convert amounts between different currencies such as US Dollars (USD), Indian Rupees (INR), and Euros (EUR). The program provides a menu-driven interface where users can choose the type of conversion they want to perform.
The application takes user input for the amount and performs real-time calculations using predefined conversion rates. It supports multiple conversion operations including USD to Rupee, Rupee to USD, Euro to Rupee, and Rupee to Euro. The result is then displayed to the user in a clear and understandable format.
This project demonstrates the use of basic Java concepts such as conditional statements, methods (functions), user input handling using Scanner, and arithmetic operations.
Key Features:
Menu-driven user interface
Supports multiple currency conversions
Simple and easy-to-use design
Uses functions for modular programming
Instant calculation and output display
--Technologies Used:
Java (Core Java)
Scanner class for input handling
--Objective:
The main objective of this project is to understand and implement basic programming concepts in Java while building a practical application that performs real-world calculations.
